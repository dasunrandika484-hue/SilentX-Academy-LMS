# 🎓 SilentX Academy — Learning Management System (LMS)

A modern, cloud-based Learning Management System designed for **Batch 2025A**, providing a seamless educational experience for students from **Grade 6 to 11**.

---

## 🚀 Features
* **Secure Authentication:** Role-based login system for Students and Admins.
* **Grade-Specific Filtering:** Students only see study materials relevant to their specific grade (Grade 6-11).
* **Admin Control Center:** Effortless management of study materials and student registrations directly via the UI.
* **Cloud Integration:** Real-time data synchronization using Firebase Firestore.
* **Responsive UI:** Fully optimized for both desktop and mobile devices.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Modern UI with Syne & DM Sans fonts), Vanilla JavaScript.
* **Backend:** Firebase (Firestore Database).
* **Hosting:** Vercel.
